# Fruit Brick Breaker

Browser-based Fruit Brick Breaker game with a portrait arcade layout inspired by classic mobile fruit puzzles.

Features:

- Modern menu and polished UI
- Multiple game modes (`Classic`, `Time Attack`, `Zen`)
- Level select and per-level intro screen
- Persistent highscores (`localStorage`)
- Background music with `Pixel Juice Power-Up`
- Generated SVG assets for cartoon fruit bricks and HUD icons

## Run

Use any static server in this folder.

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Controls

- Move paddle: `A/D` or `Arrow Left/Right`
- Launch ball: `Space` or click/tap on canvas
- Pause/Resume: `P`

## Asset Generation

Assets are generated by:

```bash
./tools/generate_assets.sh
```

This writes SVG files into:

- `assets/fruits/`
- `assets/ui/`
